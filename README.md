## Horse Race Simulator
- File: `horse_race_simulator.cpp`
- Build (mac/linux): `g++ -std=c++17 -O2 -o horse horse_race_simulator.cpp && ./horse`
- Build (Windows, MinGW): `g++ -std=c++17 -O2 -o horse.exe horse_race_simulator.cpp && horse.exe`
